# Theme Concept: Fjord Terminal

The original TallTechGuy theme — deep fjord-blue background, warm amber accent,
a terminal-window hero (`whoami` / `status`), Space Grotesk + IBM Plex Mono.
This was the live theme before Earthy Blue replaced it at the repo root.

This folder is **not live**. It's excluded from the build (see
`_theme-concepts` in the root `_config.yml`), so nothing here affects the
current site until you promote it.

## To make this the live theme again

1. Copy everything from this folder (except this README) up to the repo
   root, overwriting the current `_layouts/`, `_includes/`, `assets/`, and
   page files.
2. Push. GitHub rebuilds automatically.
